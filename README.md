# 施工報告產生器 📋

一鍵產生專業 Word 格式施工報告，適用於消防、水電、裝修等工程公司。

## 🌐 線上使用

直接開啟網頁即可使用：**[點此使用](https://你的帳號.github.io/construction-report-generator/)**

> 無需安裝任何軟體，支援電腦與手機瀏覽器

## ✨ 功能特色

- 📄 **自動排版** - 封面頁 + 照片頁，專業美觀
- 🖼️ **照片管理** - 拖曳排序、批次上傳
- 🎨 **公司 Logo** - 可上傳自訂 Logo
- 📱 **跨平台** - 電腦、平板、手機都能用
- 🔒 **隱私安全** - 所有資料僅在您的瀏覽器處理，不會上傳至伺服器

## 📖 使用說明

### 1️⃣ 填寫工程資訊
- 工程名稱（必填）
- 報告日期
- 施工說明

### 2️⃣ 上傳照片
- 點擊上傳區或直接拖曳照片
- 支援 JPG、PNG 格式
- 可多選批次上傳

### 3️⃣ 編輯照片說明
- 為每張照片填寫說明文字
- 拖曳或使用箭頭調整順序

### 4️⃣ 產生報告
- 點擊「產生 Word 報告」
- 自動下載 .docx 檔案

## 📐 報告格式

### 封面頁
- 公司 Logo（可選）
- 標題「施工報告書」
- 工程資訊表格
- 施工說明區塊

### 照片頁
- 頁首：Logo + 公司名稱 + 標語
- 照片表格：每頁 4 張（2×2）
- 頁尾：頁碼顯示

## 🛠️ 技術架構

- 純前端應用，無後端伺服器
- [docx.js](https://docx.js.org/) - Word 文件產生
- [FileSaver.js](https://github.com/eligrey/FileSaver.js/) - 檔案下載
